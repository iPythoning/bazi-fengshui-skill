      分析流程

      从输入到输出的七步分析管线

      1

      收集信息

      出生年月日时辰、性别、所在城市、当前住址、公司地址、特殊情况（负债/远程管理等）

      2

      推算八字喜忌

      排四柱 → 定日主旺衰 → 确定喜用神（补益日主的五行）与忌神（消耗日主的五行）

      3

      方位吉凶判断

      城市中心为原点划分八方位 → 结合自然地理修正五行 → 对照喜忌评级（大吉/吉/平/凶/大凶）

      4

      板块推荐

      按五行契合度分梯队 → 标注小区举例、五行优势、价格区间、适配理由 → 考虑经济状况调整

      5

      住址/办公地优化

      分析楼层五行与周边环境 → 提供颜色/方向/摆件/绿植化解方案 → 优先零成本/低成本

      6

      搬迁优先级排序

      P0立即改造 → P1本周优化 → P2考察准备 → P3长期落定 → P4远期迁移 → 结合流年吉凶调整时间

      7

      输出完整报告

      方位吉凶表 + 分梯队推荐表 + 优化方案表 + 优先级行动表 + 一句话总结
