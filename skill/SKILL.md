# 通用八字风水分析

## 概述

本技能为通用版八字风水分析工具，面向所有用户。用户输入出生年月日（含时辰更佳）和所在城市后，自动推算八字喜忌五行，生成个性化的风水行动指南。覆盖方位吉凶、板块推荐、住址优化、公司改造、搬迁规划五大模块。

## 执行流程

### 第一步：收集用户信息

向用户收集以下信息（缺失项可合理推断或追问）：

1. **出生年月日**：公历或农历均可，需注明。如能提供时辰更佳（用于推算完整四柱）
2. **性别**：影响大运排法
3. **所在城市**：用于方位吉凶判断
4. **当前住址**：小区名+楼层+所在区位（用于住址优化）
5. **公司/办公地**：地址+楼层（用于公司风水改造）
6. **特殊情况**：是否有负债压力、是否远程管理、创业/打工等（影响优先级排序）

### 第二步：推算八字喜忌五行

根据出生年月日推算八字四柱，分析日主旺衰，确定喜用神与忌神。推算方法详见 `references/bazi-calculation.md`。

核心逻辑：
1. 排四柱（年柱、月柱、日柱、时柱）
2. 定日主（日干）与日主旺衰
3. 分析五行力量对比（得令、得地、得势）
4. 确定喜用神（补益日主的五行）与忌神（消耗日主的五行）

### 第三步：方位吉凶判断

根据用户所在城市，按八个方位输出五行属性与吉凶评级。判断方法详见 `references/direction-analysis.md`。

核心逻辑：
1. 城市中心为原点，划分八个方位
2. 每个方位对应一种五行属性（结合自然地理：山水河流、地势高低）
3. 对照用户喜忌五行，判断每个方位的吉凶
4. 综合房价因素，给出性价比评级

### 第四步：板块推荐

按方位分梯队推荐适配板块/小区，兼顾五行契合度与房价现实。推荐逻辑详见 `references/direction-analysis.md`。

核心逻辑：
1. 优先推荐喜用神方位的板块
2. 按五行契合度分梯队（第一/第二/第三梯队）
3. 标注价格区间与适配理由
4. 考虑用户经济状况调整推荐

### 第五步：住址/办公地优化

针对用户当前住址和办公地，提供室内风水改造方案。优化方法详见 `references/optimization-guide.md`。

核心逻辑：
1. 分析楼层五行（河图数理）与用户喜忌的关系
2. 分析周边环境五行（山=土、水=水、林=木等）
3. 提供化解方案（颜色、方向、摆件、绿植等）
4. 优先零成本/低成本方案

### 第六步：搬迁优先级排序

结合用户经济状况、八字流年、实际需求，给出P0-P4行动建议。规划方法详见 `references/relocation-plan.md`。

核心逻辑：
1. 零成本/低成本改造优先（P0-P1）
2. 考察/准备次之（P2）
3. 大额支出/搬迁最后（P3-P4）
4. 结合流年吉凶调整时间建议

### 第七步：输出完整报告

按标准格式输出，模板详见 `references/output-templates.md`。

## 输出格式规范

完整报告包含以下部分：

1. **八字喜忌分析**：日主、旺衰、喜用神、忌神
2. **方位吉凶表**：方位 | 板块范围 | 五行属性 | 吉凶等级 | 性价比评级
3. **分梯队推荐表**：梯队 | 板块 | 小区举例 | 五行优势 | 价格区间 | 适配理由
4. **优化方案表**：位置 | 做法 | 目的
5. **优先级行动表**：优先级 | 行动 | 时间建议 | 理由
6. **一句话总结**：精炼概括核心建议

## 重要原则

- **实用性优先**：所有建议必须可落地，避免玄虚空泛
- **成本意识**：优先零成本/低成本方案，大额支出需标注时机
- **流年意识**：结合流年干支判断宜忌，火土年不宜大动作
- **因地制宣**：不同城市的方位五行属性不同，需结合实际地理判断
- **科学表述**：用五行生克逻辑解释，避免迷信表述
- **用户确认**：推算八字需用户确认出生信息准确性
