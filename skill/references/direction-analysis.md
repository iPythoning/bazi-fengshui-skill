# 方位五行分析与板块推荐方法

## 一、方位与五行对应关系

### 基础对应（后天八卦/洛书方位）

| 方位 | 八卦 | 五行 | 自然象征 |
|------|------|------|----------|
| 正东 | 震 | 木 | 日出方向，生发之气 |
| 东南 | 巽 | 木 | 风，木气流动 |
| 正南 | 离 | 火 | 阳气最盛 |
| 西南 | 坤 | 土 | 大地，承载 |
| 正西 | 兑 | 金 | 日落方向，收敛 |
| 西北 | 乾 | 金 | 天，刚健 |
| 正北 | 坎 | 水 | 阴气最盛，寒 |
| 东北 | 艮 | 土/水木过渡 | 山，止与始 |

### 方位五行修正因素

方位五行并非固定不变，需结合实际地理环境修正：

1. **山水修正**：
   - 方位内有山 → 增加土气
   - 方位内有江河湖海 → 增加水气
   - 方位内有大片森林/绿化 → 增加木气

2. **地势修正**：
   - 高地势 → 增加土气/金气
   - 低洼地带 → 增加水气
   - 平坦开阔 → 木气生发

3. **城市特殊修正**：
   - 城市不同区域的五行属性需结合当地地理判断
   - 如杭州正东有钱塘江（大水），则正东=木+大水，比标准对应更强

## 二、方位吉凶判断方法

### 判断流程

1. **确定城市中心**：以城市行政/商业中心为原点
2. **划分八个方位**：以原点为中心，按八卦方位划分
3. **标注各方位板块**：列出每个方位覆盖的区域/板块
4. **分析五行属性**：结合基础对应+地理修正，确定每个方位的五行
5. **对照喜忌**：与用户八字喜用神/忌神对照
6. **评级**：
   - 五行=喜用神 → 大吉
   - 五行生喜用神 → 吉
   - 五行克忌神 → 吉（间接助身）
   - 五行中性（不生不克） → 平
   - 五行=忌神 → 凶
   - 五行生忌神 → 大凶

### 吉凶等级定义

| 等级 | 标准 | 说明 |
|------|------|------|
| 大吉 | 五行=喜用神或双重补益 | 最佳选择 |
| 吉 | 五行生喜用神 | 良好选择 |
| 平 | 五行中性或克忌泄忌 | 可选但非最优 |
| 凶 | 五行=忌神之一 | 不推荐 |
| 大凶 | 五行=忌神且生另一忌神 | 最差选择 |

### 性价比评级

综合考虑五行契合度与房价：
- **最高**：五行大吉+房价洼地
- **极高**：五行大吉+房价中低
- **高**：五行吉+房价洼地
- **中等**：五行吉/平+房价中等
- **一般**：五行平+房价偏高
- **低**：五行凶+房价高

## 三、板块推荐方法

### 推荐流程

1. **筛选方位**：优先推荐喜用神方位
2. **分梯队**：
   - 第一梯队：大吉方位+房价洼地/中低
   - 第二梯队：吉方位+房价洼地
   - 第三梯队：吉/平方位+房价中等
3. **标注信息**：每个板块需列出：
   - 板块名称
   - 代表小区（3-4个）
   - 五行优势（自然地理带来的五行补益）
   - 价格区间（洼地/中等偏低/中等/偏高）
   - 适配理由（结合用户八字与实际情况）

## 四、各城市方位分析要点

### 杭州

- 正东：钱塘江（大水）+ 东方木 → 木+大水
- 东北：临平山（木）+东湖（水） → 水木过渡
- 正北：京杭大运河 → 纯水
- 正南：滨江/萧山南 → 火
- 西南：之江/转塘/富阳 → 土+火
- 正西：西湖区 → 金
- 西北：良渚/瓶窑 → 金+土
- 东南：萧山东南 → 木+火

### 其他城市通用分析方法

1. 找到城市最大的水体（江河湖海）→ 该方位增水气
2. 找到城市最大的山脉 → 该方位增土气
3. 找到城市东部（日出方向）→ 增木气
4. 找到城市南部（向阳方向）→ 增火气
5. 结合城市地势高低与水系分布综合判断

## 五、注意事项

1. **方位以城市中心为原点**，非用户当前位置
2. **五行属性需结合实际地理修正**，不可生搬硬套八卦对应
3. **房价信息具有时效性**，标注时注明参考时间
4. **推荐需考虑用户经济状况**，创业期/负债期优先推荐洼地
5. **通勤便利性也是考虑因素**，不可只看五行不顾实际
