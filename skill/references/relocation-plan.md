# 搬迁优先级规划方法

## 一、优先级框架

### P0-P4 分级体系

| 优先级 | 定义 | 成本 | 时间 | 适用场景 |
|--------|------|------|------|----------|
| P0 | 立即执行 | 零成本 | 立即 | 公司/住址内部风水改造（换颜色、换绿植、调方向） |
| P1 | 短期执行 | 低成本（数百元） | 1周内 | 住址室内优化（地垫、床品、摆件、加湿器） |
| P2 | 中期准备 | 考察零成本 | 3-12个月 | 租房考察、实地看房、对比分析 |
| P3 | 长期执行 | 大额支出 | 1-3年后 | 买房/长期租房落定 |
| P4 | 远期规划 | 中高成本 | 3年后 | 公司注册地迁移、大范围调整 |

## 二、优先级排序原则

### 1. 成本优先

- 零成本改造 > 低成本优化 > 考察准备 > 大额支出
- 创业期/负债期用户特别强调守现金流

### 2. 效果优先

- 影响范围大的改造优先（公司全楼层 > 个人卧室）
- 立即生效的改造优先（换颜色 > 等流年）

### 3. 流年配合

- 喜用神流年适合大动作（购房、搬迁）
- 忌神流年不宜大额支出，以内部改造为主
- 流年五行判断方法见 `references/bazi-calculation.md`

### 4. 实际需求

- 远程管理者：公司风水影响弱，住址风水影响强
- 本地办公者：公司与住址同等重要
- 创业期：优先低成本方案，大额支出延后
- 稳定期：可考虑购房与长期规划

## 三、流年与搬迁时机

### 流年吉凶判断

| 流年五行 | 与喜用神关系 | 宜忌 |
|----------|-------------|------|
| = 喜用神 | 大吉年 | 宜购房、搬迁、创业 |
| 生 喜用神 | 吉年 | 可行动，但力度适中 |
| = 忌神 | 凶年 | 不宜大额支出，守成为上 |
| 生 忌神 | 大凶年 | 严禁大动作，内部改造为主 |
| 中性 | 平年 | 视实际情况决定 |

### 搬迁时机建议

1. **忌神流年**（如身弱木命遇丙午火年）：
   - 不宜购房、不宜搬迁
   - 以室内风水改造为主（P0-P1）
   - 可进行考察准备（P2）

2. **喜用神流年**（如身弱木命遇壬子水年）：
   - 适宜购房、搬迁
   - 可执行P3级别的长期规划
   - 把握时机落定

3. **过渡年份**：
   - 视流年五行力量强弱决定
   - 力量强则按上述判断，力量弱则保守

## 四、各级行动模板

### P0 行动模板（立即执行）

**适用**：公司/住址内部风水改造
**成本**：零成本至极低成本
**内容**：
1. 更换绿植类型（水培↔土培，视喜用神）
2. 更换主色调（前台、墙面、装饰）
3. 调整工位/床位方向
4. 更换视频会议背景（远程管理者）
5. 调整财务位摆件

### P1 行动模板（本周执行）

**适用**：住址室内优化
**成本**：低成本（数百元内）
**内容**：
1. 门口地垫+门牌装饰（喜用神颜色）
2. 床品/窗帘更换（喜用神颜色）
3. 床头方向调整（朝喜用神方位）
4. 加湿器/除湿器配置（视喜用神）
5. 喜用神五行摆件（鱼缸/绿植/金属件等）
6. 化煞物品（六帝钱等）

### P2 行动模板（中期考察）

**适用**：租房/买房考察
**成本**：考察零成本
**内容**：
1. 实地考察喜用神方位的板块
2. 对比租金/房价与当前住址
3. 评估通勤便利性
4. 评估小区五行环境（周边山水、绿化）
5. 不急于签约，多看多比较

### P3 行动模板（长期落定）

**适用**：买房/长期租房
**成本**：大额支出
**内容**：
1. 等待喜用神流年（避开忌神流年）
2. 选择喜用神方位的板块
3. 选择喜用神五行楼层
4. 按风水标准选房（户型、朝向、周边环境）
5. 装修按喜用神五行配色

### P4 行动模板（远期规划）

**适用**：公司注册地迁移/大范围调整
**成本**：中高成本
**内容**：
1. 等待时机（喜用神大运/流年）
2. 选择喜用神方位的办公地
3. 按风水标准装修
4. 注册地同步变更
5. 团队搬迁安排
