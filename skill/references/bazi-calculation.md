# 八字推算与五行分析方法

## 一、四柱排法

### 1. 年柱推算

根据出生年份查天干地支：

**天干**（公历尾数）：
- 0/1 → 庚辛
- 2/3 → 壬癸
- 4/5 → 甲乙
- 6/7 → 丙丁
- 8/9 → 戊己

**地支**（生肖）：
- 子鼠、丑牛、寅虎、卯兔、辰龙、巳蛇
- 午马、未羊、申猴、酉鸡、戌狗、亥猪

**注意**：年柱以立春为界，非农历正月初一。立春前出生者属上一年。

### 2. 月柱推算

月柱地支固定：寅（正月）、卯（二月）、辰（三月）、巳（四月）、午（五月）、未（六月）、申（七月）、酉（八月）、戌（九月）、亥（十月）、子（十一月）、丑（十二月）。

月柱天干根据年干推算（五虎遁）：
- 甲己年 → 丙寅起
- 乙庚年 → 戊寅起
- 丙辛年 → 庚寅起
- 丁壬年 → 壬寅起
- 戊癸年 → 甲寅起

**注意**：月柱以节气为界，非农历初一。

### 3. 日柱推算

日柱需查万年历。公历日期对应的天干地支可通过万年历查询。

如无万年历，可使用以下Python代码片段辅助查询：
```python
# 建议使用 lunar-python 或 sxtwl 库进行精确推算
# pip install lunar-python
from lunar_python import Solar
solar = Solar.fromYmdHms(year, month, day, hour, minute, second)
lunar = solar.getLunar()
bazi = lunar.getEightChar()
# bazi.getYear() / getMonth() / getDay() / getTime()
```

### 4. 时柱推算

时柱地支按时辰：
- 子时 23:00-01:00 | 丑时 01:00-03:00 | 寅时 03:00-05:00
- 卯时 05:00-07:00 | 辰时 07:00-09:00 | 巳时 09:00-11:00
- 午时 11:00-13:00 | 未时 13:00-15:00 | 申时 15:00-17:00
- 酉时 17:00-19:00 | 戌时 19:00-21:00 | 亥时 21:00-23:00

时柱天干根据日干推算（五鼠遁）：
- 甲己日 → 甲子起
- 乙庚日 → 丙子起
- 丙辛日 → 戊子起
- 丁壬日 → 庚子起
- 戊癸日 → 壬子起

## 二、日主旺衰判断

### 1. 日主定义

日柱天干即为日主，代表命主自身。日主五行属性：
- 甲乙 → 木（甲为阳木，乙为阴木）
- 丙丁 → 火（丙为阳火，丁为阴火）
- 戊己 → 土（戊为阳土，己为阴土）
- 庚辛 → 金（庚为阳金，辛为阴金）
- 壬癸 → 水（壬为阳水，癸为阴水）

### 2. 旺衰判断三要素

**得令（月令）**：日主生于何月令
- 木：旺于寅卯月（春），相于亥子月（冬）
- 火：旺于巳午月（夏），相于寅卯月（春）
- 土：旺于辰戌丑未月（四季月），相于巳午月（夏）
- 金：旺于申酉月（秋），相于辰戌丑未月（四季月）
- 水：旺于亥子月（冬），相于申酉月（秋）

**得地（地支）**：四柱地支中有无日主的根
- 如日主为木，地支有寅、卯、辰（木之根）则为得地

**得势（天干）**：四柱天干中有无比劫（同五行）
- 如日主为木，天干有甲、乙则为得势

### 3. 旺衰结论

- 得令+得地+得势 → 身旺
- 得令+得地/得势 → 身偏旺
- 失令但得地得势 → 身中和偏旺
- 失令且仅得一项 → 身偏弱
- 失令且不得地不得势 → 身弱

## 三、喜用神与忌神确定

### 1. 基本原则

- **身旺**：喜克泄耗（官杀、食伤、财星），忌生扶（印星、比劫）
- **身弱**：喜生扶（印星、比劫），忌克泄耗（官杀、食伤、财星）

### 2. 五行喜忌对照

**日主为木**：
- 身旺：喜火（泄）、土（耗）、金（克），忌水（生）、木（助）
- 身弱：喜水（生）、木（助），忌火（泄）、土（耗）、金（克）

**日主为火**：
- 身旺：喜土（泄）、金（耗）、水（克），忌木（生）、火（助）
- 身弱：喜木（生）、火（助），忌土（泄）、金（耗）、水（克）

**日主为土**：
- 身旺：喜金（泄）、水（耗）、木（克），忌火（生）、土（助）
- 身弱：喜火（生）、土（助），忌金（泄）、水（耗）、木（克）

**日主为金**：
- 身旺：喜水（泄）、木（耗）、火（克），忌土（生）、金（助）
- 身弱：喜土（生）、金（助），忌水（泄）、木（耗）、火（克）

**日主为水**：
- 身旺：喜木（泄）、火（耗）、土（克），忌金（生）、水（助）
- 身弱：喜金（生）、水（助），忌木（泄）、火（耗）、土（克）

### 3. 十神关系

以日主为基准：
- **比肩/劫财**：同五行（助身）
- **食神/伤官**：日主所生（泄身）
- **偏财/正财**：日主所克（耗身）
- **七杀/正官**：克日主者（克身）
- **偏印/正印**：生日主者（生身）

## 四、流年五行判断

根据流年天干地支判断当年五行旺衰：

| 天干 | 五行 | 地支 | 五行 |
|------|------|------|------|
| 甲乙 | 木 | 寅卯 | 木 |
| 丙丁 | 火 | 巳午 | 火 |
| 戊己 | 土 | 辰戌丑未 | 土 |
| 庚辛 | 金 | 申酉 | 金 |
| 壬癸 | 水 | 亥子 | 水 |

**吉凶判断**：流年五行与喜用神一致则为吉年，与忌神一致则为凶年。

## 五、注意事项

1. **时辰重要性**：无时辰则缺时柱，推算精度降低，需告知用户
2. **真太阳时**：出生时间建议换算为真太阳时（经度修正），但一般分析可忽略
3. **节气界限**：月柱以节气为准，非农历月份
4. **特殊格局**：从格、化格等特殊格局需特殊分析，不适用常规旺衰判断
5. **建议使用工具**：推荐使用 `lunar-python` 等 Python 库进行精确推算
